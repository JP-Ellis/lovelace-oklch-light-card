# OkLCh Light Card

A perceptually-uniform color picker for Home Assistant lights. Pick colors on a
2D chroma×hue plane in the OkLCh color space — gradients feel even, hue rotation
is uniform, and out-of-gamut regions are visually faded.

Add a card with:

```yaml
type: custom:oklch-light-card
entity: light.living_room
```

See the full README on the [GitHub repo](https://github.com/JP-Ellis/lovelace-oklch-light-card)
for screenshots and configuration options.
